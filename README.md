# Student-Info-System
You need 3 pre-made info files with the following schemas:
* **Students**: <studID, name, address, phoneNumber>
* **Courses**: <id, cournam, courno, credits, desc>
* **Grades**: <studID, courseID, semester, grade>

7 functions:
1. Display Courses
3. Display Students
4. Display Grades
5. Display Student Record
6. Calculate GPA
7. Course Average
8. Exit
